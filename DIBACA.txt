Script PHP digunakan untuk membuat aplikasi manajemen peminjaman buku dengan menggunakan HTML, CSS (Bootstrap), dan JavaScript (sedikit) di dalamnya. Bahasa yang digunakan adalah PHP, dengan beberapa library dan fungsi bawaan PHP seperti mysqli untuk interaksi dengan database MySQL.

Berikut adalah komponen-komponen utama dalam script ini:

Server-Side Scripting: 
Dalam skrip ini, data dikirimkan melalui HTTP methods seperti GET, POST, PUT (digunakan untuk mengupdate data), dan DELETE untuk menghapus data. Ini mencerminkan prinsip dasar dari RESTful API.

PHP Native Functions:
mysqli_*: Untuk koneksi dan eksekusi query terhadap database MySQL.
header(): Untuk mengatur header HTTP, seperti redirect atau jenis konten yang dikirimkan.
isset(), empty(): Untuk memeriksa apakah suatu variabel sudah di-set atau tidak.
json_encode(): Untuk mengubah data menjadi format JSON.
HTTP Response Codes: Digunakan untuk memberikan respons HTTP yang sesuai. seperti 200 (OK), 400 (Bad Request), 404 (Not Found), 500 (Internal Server Error), dan lainnya.

HTML dan Bootstrap: Untuk membangun antarmuka pengguna (UI):
<!DOCTYPE html> dan struktur dasar HTML untuk markup halaman web.
Bootstrap CSS (<link> tag) digunakan untuk tata letak responsif dan gaya visual.
Penggunaan kelas-kelas Bootstrap seperti card, form-control, btn, dll., untuk mempercantik tampilan.

JavaScript:
document.addEventListener(): Untuk menambahkan event listener saat DOM (Document Object Model) sudah dimuat, seperti membatasi input hanya menerima angka pada field yang berhubungan dengan nomor.
$('#successModal').modal('show') dan $('#errorModal').modal('show'): Penggunaan jQuery untuk menampilkan modals jika terdapat pesan sukses atau pesan error.

SQL Query: Digunakan untuk berinteraksi dengan database MySQL.
SELECT, INSERT, UPDATE, DELETE: Untuk melakukan operasi CRUD (Create, Read, Update, Delete) terhadap data peminjaman buku.
Penanganan hasil query menggunakan mysqli_query(), dan fetch data menggunakan mysqli_fetch_array().

Fitur Aplikasi:
Form Input: Untuk menambahkan atau mengubah data peminjaman buku.
Tabel Data: Untuk menampilkan data peminjaman buku beserta aksi untuk mengedit dan menghapus data.
Validasi Input: Pengecekan agar data yang dimasukkan lengkap sebelum disubmit ke database.
Feedback Pengguna: Menampilkan pesan sukses atau pesan error setelah operasi CRUD.
